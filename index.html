Pertanyaan Umum

Jawaban untuk pertanyaan yang sering diajukan

Bagaimana cara membeli paket VPN?
Anda dapat membeli paket VPN dengan menghubungi admin kami melalui WhatsApp. Pilih paket yang Anda inginkan, lalu klik tombol "Order" untuk langsung terhubung dengan admin.
Apakah kuota VPN bisa digunakan di semua daerah?
Sebagian besar paket kami support semua daerah (ALL TKP), kecuali untuk beberapa paket tertentu seperti TSEL ILMUPEDIA yang hanya support Jateng & Jatim.
Berapa lama proses aktivasi paket?
Proses aktivasi biasanya sangat cepat, hanya membutuhkan waktu beberapa menit setelah pembayaran dikonfirmasi.
Apakah ada garansi untuk paket yang dibeli?
Untuk paket resmi dari provider, ada garansi sesuai dengan kebijakan provider masing-masing. Untuk paket unlimited unofficial, tidak ada garansi.
Aplikasi VPN apa yang direkomendasikan?
Kami merekomendasikan aplikasi HTTP Custom untuk sebagian besar paket kami. Namun, kami juga menyediakan konfigurasi untuk aplikasi VPN lain seperti V2box & Npv Tunnel (untuk iphone), Dark Tunnel, Armod, Netmod, E-V2ray, V2rayNG, HTTP Injector dan lainnya.
Bagaimana cara menghubungi admin jika ada kendala?
Anda dapat menghubungi admin kami melalui WhatsApp di nomor 6281214726068. Tim support kami siap membantu Anda 24 jam.
